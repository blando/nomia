# Contributing to Nomia

Thank you for your interest in helping improve Nomia! This repository is specifically for tracking bugs and feature requests for the [Nomia app](https://nomiaapp.com).

## How to Contribute

### Reporting Bugs

Before creating a bug report:
1. **Search existing issues** to see if the bug has already been reported
2. **Check the app** to ensure you're using the latest version
3. **Verify the bug** can be reproduced consistently

When creating a bug report, please use the bug report template and include:
- Clear, descriptive title
- Detailed steps to reproduce
- Expected vs. actual behavior
- Screenshots or recordings when applicable
- Your environment (device, OS, browser, app version)

### Requesting Features

Before submitting a feature request:
1. **Search existing requests** to avoid duplicates
2. **Consider the scope** - does it fit Nomia's purpose?
3. **Think about use cases** - who would benefit and how?

When creating a feature request, please use the feature request template and include:
- Clear description of the feature
- The problem it solves
- How you envision it working
- Any relevant examples or mockups

## Code of Conduct

- Be respectful and constructive
- Stay on topic
- No spam or self-promotion
- Follow GitHub's [Community Guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines)

## Questions?

If you have questions about using Nomia, please visit [nomiaapp.com](https://nomiaapp.com) for support resources.
